## 🛠️ Tecnologias e Conceitos Aplicados

Neste projeto, foquei em demonstrar as seguintes competências:

* **CSS Grid & Flexbox:** Utilizados para criar uma malha de serviços responsiva e um cabeçalho dinâmico e organizado.
* **UI Industrial:** Design focado em legibilidade e seriedade, utilizando uma paleta de cores técnica (Steel Grey e Warning Orange).
* **Navegação Sticky:** Implementação de menu fixo para melhorar a experiência do usuário em páginas de rolagem única (Single Page).
* **Conversão via WhatsApp:** Botão flutuante/estratégico configurado para facilitar a solicitação de orçamentos em tempo real.
